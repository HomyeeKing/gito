# gito

A CLI to expand the ability of `git`.

# install

`cargo install gito`

# Commands

## gito get-upstream

or `gito gup` to auto set the upstream ssh-url if the repo is forked:

w/ options:

```bash
--remote-name [name] # default is `upstream`
```
